# Fault-Tolerant Distributed Key-Value Store (Raft)

A distributed key-value store built using the Raft consensus algorithm.

## 🚀 Highlights

* 50,000+ ops/sec
* Sub-5ms replication latency
* Leader election & fault tolerance

## ⚙️ Tech Stack

* Java
* gRPC
* Docker
* AWS

## ▶️ Run

docker-compose up --build
